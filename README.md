# Real-Time Smoking Detection using YOLOv11 and MediaPipe Holistic Fusion

## Description
This project is an AI-based system that detects smoking activity in real time using computer vision techniques. It uses YOLOv11 for object detection and MediaPipe for feature analysis.

## Technologies Used
- Python
- YOLOv11
- MediaPipe
- OpenCV

## Features
- Real-time smoking detection
- Uses deep learning models
- Practical application for monitoring systems

## Project Objective
To build an intelligent system that can automatically detect smoking behavior and improve monitoring efficiency.

## Status
Completed as an academic project

- Improve detection accuracy
